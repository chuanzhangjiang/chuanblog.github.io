# Nerd字体全量安装文档

## clone Nerd仓库
```
git clone https://github.com/ryanoasis/nerd-fonts.git
```

## 运行`./install.sh`文件

```
cd nerd-fonts

sudo chmod u+x install.sh

./install.sh
```
